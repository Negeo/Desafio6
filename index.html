Mis Intereses

Soy un apasionado de la tecnología. Desde pequeño me gustaba desarmar radios y juguetes y aprender cómo funcionaban. Ahora me encamino hacia la ciberseguridad y al desarrollo web, dos campos que me parecen muy interesantes y desafiantes. Me gusta estar al día de las últimas novedades y tendencias en seguridad informática y programación. También disfruto de las artes marciales, cocinar y llegar caminando a la máxima cantidad de lugares posbiles. Proyectos

HTML

HTML significa HyperText Markup Language, que significa lenguaje de marcado de hipertexto. HTML consiste en etiquetas que definen la estructura y el contenido de una página web, como los encabezados, los párrafos, las imágenes, los enlaces, etc. Para ver una página web creada con HTML, se necesita un navegador web que interprete el código HTML y lo muestre en la pantalla.

Ver más

CSS

Con CSS, se puede cambiar el color, la fuente, el tamaño, el fondo, el borde y la posición de los elementos HTML. CSS también permite crear animaciones, transiciones y efectos visuales. CSS se escribe en un archivo separado del HTML, y se enlaza con la etiqueta link en el head de la página. Así, se puede modificar el aspecto de toda la página sin tener que cambiar el HTML.

Ver más

BOOTSTRAP

Bootstrap es un framework de diseño web que te permite crear sitios web responsivos y adaptables a diferentes dispositivos. Con Bootstrap, puedes usar componentes predefinidos como botones, menús, formularios, tarjetas, etc. para facilitar el desarrollo de tu página web. Bootstrap también incluye un sistema de cuadrícula que te ayuda a organizar el contenido de tu sitio web de forma flexible y consistente.
